# SMTP-Client-Express
A simple website with a panel to send emails via SMTP Servers.


# Requirements

1. `NodeJS installed`.
2. `SMTP credentials`.

# To Do

1. `Install this repository.`
2. `Run npm install`.
3. `Edit the config.json`
4. `open terminal and run node .`
